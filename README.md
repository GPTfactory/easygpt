## How to load as an unpacked extension

1. Download the code
2. Go to `chrome://extensions/`
3. Turn on developer mode (top right corner)
4. Click `Load unpacked`
5. Select the folder that contains the code `/src`
6. Enjoy (👉ﾟヮﾟ)👉
